🌸 Iris Flower Classification Using K-Nearest Neighbors (KNN)

This project demonstrates a machine learning classification model using the famous Iris dataset. The objective is to classify iris flowers into three species — Setosa, Versicolor, and Virginica — based on their petal and sepal measurements.

The project implements the K-Nearest Neighbors (KNN) algorithm, applies data preprocessing and feature scaling, evaluates multiple K values using the Elbow Method, and visualizes model performance through confusion matrices, decision boundaries, and F1-score analysis.

🚀 Features
Data preprocessing and feature scaling using StandardScaler

KNN Classification Model
Optimal K selection using Elbow Method
